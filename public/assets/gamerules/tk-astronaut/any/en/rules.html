Game Rules (9/8/2021)

Note: The Arcader video slot is referred to as Arcader The Game or Game.

1. Arcader is a video slot with 5 reels and 15 fixed pay lines. The game features wilds, expanders, mystery game and bonus game with free spins. The theoretical return to player is 96.1%.

2. A game round is started with the action button and plays the game with the selected bet level.

3. No bets can be altered during a game round.

4. Auto play plays the game automatically for the number of game rounds selected in the auto play page. This feature may not be available.

5. Expanders expands the reel it appears on and fills it with the symbols representing the best possible pay line.

6. The bonus symbol is a scatter symbol that can appear on any reel and grants the bonus game with free spins if 3 or more bonus symbols are won. The number of free spins won is according to the pay table and the number of bonus symbols won grants a certain multiplier according to the pay table. Each bonus game free spin grants the chance of a new sticky wild on the three central reels. The sticky wilds remain sticky throughout the bonus game.

7. The mystery symbol is a scatter symbol that can appear on any reel and grants the mystery game with mystery wins if 3 or more mystery symbols are won. The number of mystery symbols won grants a certain multiplier according to the pay table. The mystery game ends either by selection of a blank or by the selection of all mystery wins.

8. Wilds and sticky wilds substitutes all other symbols except expanders, mystery symbols and bonus symbols.

9. All game payout and win combinations are paid out according to the pay table.

10. Only the highest win per pay line is paid out, if in sequence from far left to right in combinations according to the pay table.

11. All pay line wins during a game round are added and presented at the end of the game round.

12. All wins are presented and paid out in the selected currency.

13. The bonus game and mystery game is played with the same bet as the spin that triggered the bonus game or mystery game.

14. In the event of game malfunction all affected bets and pays are rendered void.
